# Pipelines

Pipelines define sequential and cyclical processes within the IMAGHOSTNOW system.

## 1. Signal Evaluation Loop

Stages:
1. intake  
2. validation  
3. alignment  
4. weighted interpretation  
5. state update  
6. coherence verification  

Rules:
- output must match structure shape
- invalid signals trigger reconciliation

## 2. Coherence Propagation Pipeline

Steps:
1. root invariants anchor the flow
2. framework law distributes boundaries
3. algorithmic logic transforms meaning
4. architecture channels route signals
5. data layer maps outcomes
6. visual layer renders symbolic forms
7. creative layer transforms into expression

## 3. Reconciliation Cycle

Triggered when:
- drift detected  
- violated invariant  
- shape mismatch  
- flow inconsistency  

Cycle steps:
1. isolate mismatch  
2. trace upstream source  
3. reapply layer constraints  
4. re-evaluate logic  
5. re-align structure  
6. stabilize global coherence  

Pipelines ensure no ambiguity exists in state propagation or system evolution.
